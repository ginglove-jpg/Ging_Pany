# 신년계획서 사이트 (방 링크 공유형)

## 핵심 흐름
1) **보스(주인장)**: 닉네임/비번 입력 → **Start(방 생성)**
2) 생성된 링크 `/r/XXXXXXXXXX` 를 다른 사람들에게 공유
3) **참가자**: 공유된 링크에서 닉네임/비번으로 로그인 → **작성 저장**
4) 모두 작성 끝났으면 **보스만** `종료(365일 잠금)`

## 기능
- **간단 로그인(닉네임/비번)** 기반 (공유 IP 환경에서도 다중 사용자 가능)
- **닉네임당 1회 작성** (삭제해도 이번 사이클 재작성 불가)
- 수정/삭제 가능 (단, 보스가 종료하기 전까지)
- **전체 공개 목록**: 잠금 전에는 모든 작성글이 목록으로 공개 표시
- **보스 전용**
  - 테마(색/문구/폰트) 꾸미기
  - 종료 버튼(보스에게만 노출)
- **종료(잠금) 규칙**
  - 종료하면 글쓰기/수정/삭제 모두 불가
  - 사이트 화면에 **종료 일자/시간만 표시**
  - 365일 뒤 자동으로 새 사이클 오픈 (보스/유저/작성글 초기화)

## 실행 방법 (Windows)
1) Node.js 설치 (LTS 권장)
2) 이 폴더에서 터미널(cmd/PowerShell) 열기
3) 아래 명령어:
```bash
npm install
npm start
```
4) 브라우저:
- http://localhost:3000

## 옵션 (선택)
- 데이터 파일 위치 변경:
  - `.env`에 `DATA_FILE=./data.json` 처럼 지정 가능
